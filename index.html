Common CSS Properties

The Properties

Tag Name Description
<color> Color Sets the color of a text
<background-image> Background Image Specifies one or more background images for an element
<border> Border A shorthand property for border-width, border style and border-color
<font-family> Font Family Specifies the font family for text
<height> Height Sets the height of an element
<letter-spacing> Letter Spacing Increases or decreases the space between characters in a text
<text-align> Text Align Specifies the horizontal alignment of a text
<text-decoration> Text Decoration Specifies the decoration added to text
